Qu'est-ce que Kubii Shield ?

Kubii Shield est une solution complète de protection de données pour les utilisateurs particuliers, basée sur un Raspberry Pi. Elle intègre des services essentiels comme un VPN, un bloqueur de publicités, un pare-feu et un scanner USB pour protéger votre vie privée et vos données.

Configuration initiale

Pour commencer avec votre Kubii Shield :

  1. Connectez votre Kubii Shield à votre routeur avec un câble Ethernet
  2. Branchez l'alimentation et attendez que l'appareil démarre
  3. Connectez-vous à l'interface web à l'adresse http://kubii-shield.local ou via l'adresse IP de l'appareil
  4. Suivez l'assistant de configuration pour paramétrer votre réseau et activer les services souhaités

Tableau de bord

Le tableau de bord vous offre une vue d'ensemble de l'état de votre Kubii Shield :

Services disponibles

Service VPN

Le service VPN vous permet de naviguer de manière anonyme et sécurisée sur Internet :

  • Sélection de serveurs : Choisissez parmi différents serveurs dans le monde
  • Kill Switch : Protège vos données en cas de déconnexion VPN
  • Protocoles : Support pour WireGuard (recommandé) et OpenVPN
Vidéo : Comment configurer le VPN

Anti-Publicités

Le bloqueur de publicités filtre les publicités et trackers au niveau du réseau :

  • Listes de blocage : Gérez les listes utilisées pour le filtrage
  • Statistiques : Visualisez le nombre de publicités bloquées
  • Personnalisation : Ajoutez vos propres règles de blocage
Afficher les listes recommandées

Listes recommandées :

  • EasyList : Blocage général des publicités
  • EasyPrivacy : Protection contre le pistage
  • AdGuard Base : Complémentaire à EasyList
  • Liste spécifique à votre langue

Pare-feu

Le pare-feu protège votre réseau contre les intrusions :

  • Règles personnalisées : Créez des règles pour autoriser ou bloquer le trafic
  • Blocage géographique : Filtrez le trafic par pays d'origine
  • Détection d'intrusion : Protection contre les scans de ports et attaques

Scan USB

Le scanner USB vous permet d'analyser vos périphériques contre les malwares :

  • Scan automatique : Détection et analyse automatique des périphériques connectés
  • Quarantaine : Isolation des fichiers suspects
  • Historique : Consultez l'historique des scans précédents

Problèmes de connectivité

Je ne peux pas accéder à l'interface web

Si vous ne pouvez pas accéder à l'interface web :

  1. Vérifiez que votre Kubii Shield est allumé (LED verte allumée)
  2. Assurez-vous que le câble Ethernet est correctement branché
  3. Essayez d'accéder via l'adresse IP plutôt que le nom d'hôte
  4. Redémarrez votre Kubii Shield et votre routeur

Si le problème persiste, vous pouvez réinitialiser les paramètres réseau en maintenant le bouton de réinitialisation pendant 5 secondes.

Internet ne fonctionne pas quand le VPN est activé

Si vous rencontrez des problèmes de connexion avec le VPN activé :

  1. Essayez de changer de serveur VPN
  2. Vérifiez que votre connexion Internet fonctionne sans le VPN
  3. Essayez de changer de protocole (WireGuard ou OpenVPN)
  4. Désactivez temporairement le Kill Switch dans les paramètres VPN
Le bloqueur de publicités bloque trop de sites

Si certains sites web légitimes sont bloqués :

  1. Allez dans la section Anti-Publicités
  2. Ajoutez le domaine à la liste des exceptions
  3. Considérez la désactivation temporaire du blocage pour ce site
  4. Vérifiez quelles listes de blocage sont trop agressives et désactivez-les

Problèmes de performance

L'appareil est lent ou surchauffe

Si votre Kubii Shield est lent ou chauffe anormalement :

  1. Vérifiez que l'appareil est dans un endroit bien ventilé
  2. Désactivez les services que vous n'utilisez pas
  3. Vérifiez l'utilisation CPU et RAM dans le tableau de bord
  4. Considérez un redémarrage du système
L'appareil est lent ou surchauffe

Si votre Kubii Shield est lent ou chauffe anormalement :

  1. Vérifiez que l'appareil est dans un endroit bien ventilé
  2. Désactivez les services que vous n'utilisez pas
  3. Vérifiez l'utilisation CPU et RAM dans le tableau de bord
  4. Considérez un redémarrage du système
Internet est plus lent avec le VPN

Une réduction de la vitesse avec un VPN est normale, mais pour l'optimiser :

  1. Choisissez un serveur géographiquement proche de vous
  2. Utilisez le protocole WireGuard qui est plus rapide qu'OpenVPN
  3. Vérifiez votre vitesse de connexion sans le VPN pour comparaison
  4. Essayez différents serveurs aux heures de pointe

Que fait exactement Kubii Shield ?

Kubii Shield est une solution tout-en-un pour protéger votre vie privée et vos données sur Internet. Il s'agit d'un appareil basé sur Raspberry Pi qui intègre plusieurs services de sécurité :

  • Un VPN pour naviguer anonymement
  • Un bloqueur de publicités au niveau du réseau
  • Un pare-feu pour protéger contre les intrusions
  • Un scanner USB pour détecter les malwares

Ces services protègent tous les appareils connectés à votre réseau sans avoir à installer de logiciel sur chacun d'eux.

Comment mettre à jour mon Kubii Shield ?

Kubii Shield se met à jour automatiquement par défaut. Vous pouvez vérifier manuellement les mises à jour depuis le tableau de bord en cliquant sur "Rechercher des mises à jour". Si vous préférez, vous pouvez configurer les paramètres de mise à jour dans la section Paramètres > Mises à jour.

Le VPN fonctionne-t-il sur tous mes appareils ?

Oui, lorsque le VPN est activé sur votre Kubii Shield, tout le trafic Internet passant par votre réseau est protégé. Cela inclut vos ordinateurs, smartphones, tablettes, TV connectées et tout autre appareil connecté à votre réseau domestique. Vous pouvez également configurer des règles pour exclure certains appareils du tunnel VPN si nécessaire.

Un site que j'utilise ne fonctionne plus avec le bloqueur de publicités

Certains sites peuvent ne pas fonctionner correctement avec le bloqueur de publicités actif. Vous avez plusieurs options :

  1. Ajouter le domaine à la liste des exceptions dans la section Anti-Publicités
  2. Désactiver temporairement le bloqueur de publicités
  3. Ajuster les listes de blocage utilisées pour être moins restrictives
Le scanner USB peut-il détecter tous les virus ?

Le scanner USB de Kubii Shield utilise ClamAV, une solution antivirus open source. Il peut détecter la plupart des malwares connus, mais comme tout antivirus, il n'est pas infaillible face aux menaces très récentes ou complexes. Pour une protection optimale :

  • Gardez votre Kubii Shield à jour pour avoir les dernières définitions de virus
  • Utilisez le mode de scan approfondi pour les fichiers sensibles
  • N'utilisez pas de périphériques USB provenant de sources non fiables
Quelle est la différence entre WireGuard et OpenVPN ?

WireGuard et OpenVPN sont deux protocoles VPN disponibles sur Kubii Shield :

  • WireGuard : Plus récent, plus léger et généralement plus rapide. Recommandé pour la plupart des utilisateurs.
  • OpenVPN : Plus établi, très sécurisé mais peut être plus lent. Meilleure option en cas de problèmes de compatibilité avec WireGuard.

Vous pouvez changer de protocole à tout moment dans les paramètres VPN en fonction de vos besoins.

Procédure de mise à jour

Pour garantir le bon fonctionnement de votre Kubii Shield, nous vous recommandons de maintenir votre système à jour :

1

Vérifier les mises à jour

Depuis le tableau de bord, cliquez sur "Rechercher des mises à jour" ou accédez à Paramètres > Mises à jour.

2

Sauvegarder votre configuration

Avant toute mise à jour majeure, sauvegardez votre configuration depuis Paramètres > Sauvegarde > Exporter.

3

Installer la mise à jour

Cliquez sur "Installer" et attendez que le processus se termine. L'appareil redémarrera automatiquement.

4

Vérifier après la mise à jour

Après le redémarrage, vérifiez que tous vos services fonctionnent correctement.

Important

Ne coupez jamais l'alimentation de votre appareil pendant une mise à jour. Cela pourrait endommager le système et nécessiter une réinstallation complète.
